# Guía de Nombramiento de Imágenes - Almah Landing

Para que las imágenes carguen correctamente en la aplicación, debes guardarlas en la carpeta `src/assets/` con los siguientes nombres exactos:

## Sección: Sobre nosotros (About Us)
- `about-hero.png`: Imagen principal del equipo trabajando (Hero).
- `team-nicolas.png`: Foto de Nicolás Lozada (CEO).
- `team-jacobo.png`: Foto de Jacobo Gomez (CTO).
- `team-juan.png`: Foto de Juan Camilo Girón (Coordinador jurídico).
- `team-shirly.png`: Foto de Shirly Plaza (CAO).

## Sección: Servicios (Services)
- `services-preview.png`: Imagen del banner principal mostrando la interfaz de los servicios.
- `service-communities.png`: Imagen para "Comunidades profesionales".
- `service-docs.png`: Imagen para "Explorador y vista de documentos".
- `service-chatbot.png`: Imagen para "Chatbot con citas".
- `service-ai-tools.png`: Imagen para "Herramientas de IA enfocadas".

## Sección: Comunidades (Home)
- `community-privacy.png`: Imagen para la tarjeta "Hub de Privacidad".
- `community-fintech.png`: Imagen para la tarjeta "Fintech School".

---
**Nota:** Se recomienda usar formato `.png` o `.jpg` optimizado para web. Si usas una extensión diferente, asegúrate de actualizar la importación en el componente correspondiente.
